# Plantes-database

Practice repository for learning Git workflows

# Database Description.
This repository contains organized information about planets in our solar system, including key facts and basic planetary data such as distance from the Sun, diameter, and number of moons. It highlights characteristics of both inner and outer planets, notable features like extreme temperatures, planetary rings, and long-lasting storms, and provides simple datasets for comparison.

# Fun Fact: 
Venus is hotter than Mercury even though Mercury is closer to the Sun
